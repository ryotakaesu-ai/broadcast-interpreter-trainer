# 放送通訳トレーナー / Broadcast Interpreter Trainer

NHK 水準の放送通訳学習者を対象にした、ブラウザだけで動く英語学習ゲーム。

## カテゴリー

- 🗞️ **語彙 (Vocabulary)** — 時事英単語を英⇄日双方向で
- 💬 **フレーズ (Phrases)** — `in the wake of` / `at the helm of` 等の定型表現
- 📰 **時事 (Current Events)** — ニュース文の空所補充
- 🔢 **数字 (Numbers)** — 万・億・兆 ↔ million / billion / trillion
- 🎙️ **サイトラ (Sight Translation)** — 自然な放送訳の選択

## 機能

- 制限時間モード: 15s / 10s（速報）/ 7s（同通）
- 英語プロンプトの音声読み上げ（Web Speech API）
- `1`–`4` キーで即答、`Enter` で次へ
- ストリークとハイスコアを localStorage に保存
- 終了画面に全問の振り返り

## 使い方

`index.html` をブラウザで開くだけ。インストール不要・ネット接続不要。
